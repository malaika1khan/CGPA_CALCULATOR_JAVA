# CGPA_CAlCULATOR_JAVA
A simple GUI calculator to calculate  semester GPA and Cumulative GPA with full input validation.
## Features
-**GUI Input**: JOptionPane popups instead of console.
-**Error Handling**: Handles alphabets , null/cancel , negative , >100 marks.
-**Validation loop**: User can't proceed until valid input given.
-**Formula**: CGPA=(PrevCGPA* PrevCredits + GPA*Currntcredits) / TotalCredits

## How to Run
1. Clone repo : `git clone https://github.com/USERNAME/CGPA_CALCULATOR_JAVA.git`
2. Open `MynewC.java` in NetBeans/Intelij/Eclipse
3. Run the file
4. Follow popup instructions
